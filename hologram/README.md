# Hologram — Futuristic Visual Effects

Shader-based animations, dynamic wallpapers, GLSL effects, and visual enhancements
for the Omarchy Hyprland desktop. Turns your workspace into a cyberpunk dream.

## Structure

```
hologram/
├── shaders/             # GLSL shader effects
│   ├── cyber-grid.glsl  # Holographic grid overlay
│   ├── scanlines.glsl   # CRT scanline effect
│   ├── aurora.glsl      # Northern lights ambient
│   └── matrix.glsl      # Matrix rain effect
├── wallpapers/          # Animated/dynamic wallpapers
│   ├── neon-city/       # Cyberpunk cityscape
│   ├── deep-space/      # Cosmic nebula
│   └── waveform/        # Audio-reactive visualizer
├── animations/          # Hyprland animation overrides
│   ├── cyber.conf       # Neon-border animations
│   ├── holographic.conf # Ghost/fade transitions
│   └── glitch.conf      # Glitch-effect workspace switch
└── effects/             # Desktop enhancement scripts
    ├── border-glow.sh   # Active window border pulse
    └── particle.sh      # Cursor particle effects
```
