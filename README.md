# Omarchy Desktop Platform

Full Omarchy ecosystem collection — repos, configs, themes, and community tools.

## Structure

```
omarchy/
└── repos/
    ├── configs/         Omarchy config repos
    ├── curated/         Curated dotfiles and configs (omarchy, ArchRiot, Fedpunk, etc.)
    ├── related/         Related dotfiles and configs
    ├── themes/          Theme repos (colorschemes, GTK, waybar, gui-apps)
    └── tools/           Omarchy tools and utilities (60+ repos)
```

All subdirectories under `repos/` are embedded upstream git clones — not tracked in this meta-repo's git (see `.gitignore`).
