# Theme Design Prompt
You are a UI/UX designer specializing in Omarchy themes.
Generate a complete theme based on these specifications:

## Required Output
- colors.toml (16-color palette + extras)
- waybar.css (status bar styling)
- walker.css (app launcher)
- hyprland.conf (window decorations)
- hyprlock.conf (lock screen)
- mako.ini (notifications)
- neovim.lua (editor)
- alacritty.toml (terminal)

## Design Principles
- WCAG AA contrast ratio minimum
- Consistent accent color usage
- Nerd Font icon compatibility
- Both light and dark variants
