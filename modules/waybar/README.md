# Waybar Custom Modules

Custom scripts for Waybar, extracted and refined from the ecosystem.

| Script | Description |
|--------|-------------|
| `window-pill.py` | Socket-based active window pill (class + title via hyprctl) |
| `scrolling-mpris.py` | Smooth-scrolling MPRIS now-playing module |
| `cava.sh` | Braille audio visualizer (auto-hides on silence) |
| `omarchy-logo.py` | SVG glyph generator for the Omarchy logo button |
