# 3-2-1 Backup Strategy

The industry-standard backup rule applied to Omarchy:

## The Rule
- **3** copies of your data
- **2** different storage media types
- **1** copy offsite

## Omarchy Implementation

### Copy 1: Primary (local SSD/NVMe)
```
Location: ~/.config/omarchy, ~/.config/hypr, etc.
Method: Daily btrfs snapshots (ark/snapshots/btrfs/)
Schedule: Every boot + hourly
```

### Copy 2: Secondary (external/USB/NAS)
```
Location: /mnt/backup/omarchy-$(hostname)/
Method: rsync via ark/singularity/backup/dotfiles.sh
Schedule: Daily at 3am (systemd timer)
```

### Copy 3: Offsite (cloud/remote)
```
Provider: Option A: Syncthing to home server
         Option B: BorgBase / rsync.net
         Option C: Encrypted tarball to S3
Method: ark/matrix/sync/dotfiles.sh push
Schedule: Weekly
```
